🦅 Birds API

REST API for Birds Data with JWT Authentication

Base URL

https://localhost:3001
All API requests should be made to this base URL

Authentication

Most endpoints require JWT authentication. First, obtain a token by logging in.

POST /auth/login
Login to get an access token
{ "username": "bpi", "password": "bpi2023" }
Response: { "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." }
Note: Include the token in the Authorization header for protected endpoints:
Authorization: Bearer <your_access_token>

Birds API

All bird endpoints require authentication. Include the JWT token in the Authorization header.

GET /birds Auth Required
Get all birds
Response: Array of bird objects [ { "Id": 1, "BirdMyanmarName": "ငှက်စိမ်းရင်ဝါ", "BirdEnglishName": "Orange-bellied Leafbird", "Description": "...", "ImagePath": "img/birds/img/1_Orange-belliedLeafbird.jpg" }, ... ]
GET /birds/:id Auth Required
Get a specific bird by ID
Example: GET /birds/1
POST /birds Auth Required
Create a new bird
Request Body: { "BirdMyanmarName": "ငှက်အမည်", "BirdEnglishName": "Bird Name", "Description": "Description text", "ImagePath": "img/birds/img/image.jpg" }
PUT /birds/:id Auth Required
Update a bird (full update)
Example: PUT /birds/1 Request Body: (same as POST)
PATCH /birds/:id Auth Required
Update a bird (partial update)
Example: PATCH /birds/1 Request Body: (partial fields allowed)
DELETE /birds/:id Auth Required
Delete a bird
Example: DELETE /birds/1 Response: { "message": "Bird deleted successfully" }

Example Usage

# 1. Login to get token curl -X POST https://localhost:3001/auth/login \ -H "Content-Type: application/json" \ -d '{"username":"bpi","password":"bpi2023"}' # 2. Get all birds (use token from step 1) curl -X GET https://localhost:3001/birds \ -H "Authorization: Bearer YOUR_TOKEN_HERE" # 3. Get a specific bird curl -X GET https://localhost:3001/birds/1 \ -H "Authorization: Bearer YOUR_TOKEN_HERE" # 4. Create a new bird curl -X POST https://localhost:3001/birds \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -H "Content-Type: application/json" \ -d '{ "BirdMyanmarName": "ငှက်အမည်", "BirdEnglishName": "Bird Name", "Description": "Description", "ImagePath": "img/birds/img/image.jpg" }'
